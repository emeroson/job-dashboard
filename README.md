# Job Dashboard – Offres d'Emploi en Côte d'Ivoire

Ce projet est un dashboard interactif développé avec **Python et Streamlit** pour analyser les offres d’emploi collectées sur internet.

Le projet utilise le **web scraping** pour récupérer les données et les visualiser dans un tableau de bord moderne.

## Objectif du projet

Analyser le marché de l’emploi en Côte d’Ivoire à partir des offres collectées sur des sites d’emploi.

Le dashboard permet de :

- Visualiser le nombre total d’offres d’emploi
- Identifier les entreprises qui recrutent le plus
- Analyser les villes avec le plus d’opportunités
- Voir les métiers les plus demandés
- Télécharger les données au format CSV

## Fonctionnalités du dashboard

- KPI interactifs (Total Offres, Entreprises, Postes, Villes)
- Graphiques des offres par ville
- Top entreprises qui recrutent
- Camembert de répartition des villes
- Carte interactive de la Côte d’Ivoire
- Heatmap des métiers les plus demandés
- Téléchargement des données CSV

## Technologies utilisées

- Python
- Streamlit
- Pandas
- Plotly
- Web Scraping (BeautifulSoup)
